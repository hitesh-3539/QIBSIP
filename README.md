## 🛠️ Completed Task Detail

### Task 2: BMI Calculator
A command-line tool built using Python to calculate an individual's Body Mass Index (BMI) based on their weight (kg) and height (m).

#### ✨ Features
* **User Input Validation:** Gracefully handles invalid inputs like alphabetical characters or negative values without crashing.
* **Smart Unit Conversion:** Detects if a user inputs height in centimeters (e.g., 177) instead of meters (e.g., 1.77) and automatically auto-corrects it to ensure accurate calculation.
* **Actionable Feedback:** Categorizes health metrics into Underweight, Normal weight, Overweight, and Obese with tailored, helpful health tips.
