## 🛠️ Completed Tasks Detail

### Task 2: BMI Calculator
A command-line tool built using Python to calculate an individual's Body Mass Index (BMI) based on their weight (kg) and height (m).
* **✨ Features**: Gracefully handles invalid alphabetical/negative inputs and includes smart unit conversion (automatically converts cm inputs over 3.0 down to meters).

### Task 3: Random Password Generator
A robust command-line utility that generates customizable, secure, and completely random passwords based on user preferences.
* **✨ Features**: Allows users to specify password length and select combinations of letters, numbers, and symbols. Includes built-in structural security logic ensuring at least one character from each selected pool is forced into the final result to guarantee structural strength.
